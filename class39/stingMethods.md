#	Method	What It Does
1	length	Gives total number of characters in a string (property, not a method).
2	toUpperCase()	Converts all letters to uppercase.
3	toLowerCase()	Converts all letters to lowercase.
4	charAt()	Returns the character at a specific position.
5	indexOf()	Finds the first position of a substring.
6	lastIndexOf()	Finds the last position of a substring.
7	includes()	Checks if a substring exists (true/false).
8	startsWith()	Checks if the string starts with a specific word.
9	endsWith()	Checks if the string ends with a specific word.
10	slice()	Extracts part of a string and returns it as a new one.
11	substring()	Similar to slice(), but doesnâ€™t accept negative indexes.
12	substr()	Extracts part of string by position + length (older method).
13	replace()	Replaces the first matched substring.
14	replaceAll()	Replaces all matched substrings.
15	trim()	Removes spaces from both ends.
16	trimStart()	Removes spaces only from the start.
17	trimEnd()	Removes spaces only from the end.
18	split()	Splits the string into an array (based on separator).
19	concat()	Joins two or more strings together.
20	repeat()	Repeats the string a given number of times.
21	padStart()	Pads the string from the start to reach a certain length.
22	padEnd()	Pads the string from the end.
23	toString()	Converts value to string (rarely needed for real strings).
24	valueOf()	Returns the primitive string value (used internally).
25	match()	Searches a string for pattern (often used with regex).
26	matchAll()	Returns all matches for a regex (iterator).
27	search()	Searches for a substring or regex and returns index.
28	localeCompare()	Compares two strings alphabetically.
